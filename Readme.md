# Majiang Logic Engine

这是一个基于 Python 的贵阳捉鸡麻将（27张纯净版：万、条、筒）逻辑与 AI 决策引擎。

目前项目已完成核心牌理逻辑与全阶段的出牌策略算法，主要功能包括：

## 核心功能

* **极速判胡与听牌**：基于初始化在内存中生成的组合表，实现 O(1) 复杂度的微秒级胡牌判定（内置七对特判），并精准计算听牌列表。
* **全阶段 AI 出牌推荐（分层决策架构）**：
  * **一向听精确预测**：当手牌接近听牌时，通过穷举未来的物理进张，找出能带来最多听牌机会的最优解。
  * **多向听宏观降维**：当起手牌较差（两向听及以上）时，引擎会自动切换至 DFS 状态机与向听数公式，计算手牌的最佳拆解方式（面子与搭子组合），规划出最快逼近听牌的做牌路线。

## 后续计划

本项目目前处于纯逻辑引擎阶段，使用 Mock 数据进行测试验证。下一步计划接入视觉识别模块，实现完整的端到端辅助系统：
1. 接入 YOLO 目标检测模型，实时识别真实环境中的麻将手牌。
2. 将视觉标签映射为引擎可计算的底层数组。
3. 结合本引擎的策略算法，在前端提供实时的最优打法提示。
