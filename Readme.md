# Majiang Logic Engine

这是一个基于 Python 的贵阳捉鸡麻将（27张牌：万、条、筒）逻辑引擎。

目前项目已完成核心的牌理逻辑，主要功能包括：
* 判断胡牌（包含七对等特殊牌型）
* 计算当前手牌的听牌列表
* 未听牌状态下的出牌策略推荐（支持一向听预测，最大化进张）

## 后续计划

本项目目前使用 Mock 数据进行纯逻辑测试。后续计划接入视觉识别模块，实现端到端的辅助系统：
1. 接入 YOLO 目标检测模型，实时识别真实环境中的麻将手牌。(训练数据thanks to sun yuqin)
2. 将视觉标签映射为引擎可计算的底层数组。
3. 结合本引擎的策略算法，在前端（如 AR 眼镜或手机界面）实现实时的最优打法提示。
